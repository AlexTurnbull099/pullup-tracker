# Summer Pull-Up Tracker

A personal, offline-first single-page app to track a 10-week summer pull-up program
(**Heavy → Volume → Light → Heavy → Volume → Light → Rest** cycle).

- Open `index.html` in any browser — no build step, no backend, no accounts.
- All workout data is stored on-device in the browser's `localStorage`.
- On iPhone: open the published page in Safari → **Share → Add to Home Screen** for an app-like, persistent experience.
- Export your log anytime from **Settings → Export (JSON / CSV)**.

The training program lives in [`pullup-summer-plan.md`](pullup-summer-plan.md);
the app spec is in [`pullup-tracker-brief.md`](pullup-tracker-brief.md).
